# 🍅 Pomodoro Timer

A simple **Pomodoro Timer** built with **HTML, CSS, and JavaScript** to improve focus and productivity. This project was created as a learning exercise to understand JavaScript timers, state management, and DOM manipulation without using any frameworks.

---

## 📌 Features

* ⏱️ Countdown Timer
* ▶️ Start Timer
* ⏸️ Pause Timer
* 🔄 Reset Timer
* ⏭️ Skip Current Session
* 🍅 Work Session
* ☕ Short Break
* 🌴 Long Break
* 🔢 Automatic Work Session Counter
* 🔄 Automatic Mode Switching
* 📱 Responsive UI

---

## 🛠️ Built With

* HTML5
* CSS3
* Vanilla JavaScript

No external libraries or frameworks are used.
